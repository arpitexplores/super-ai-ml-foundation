# Examples

Use these prompts to test or demonstrate `super-ai-ml-foundation`.

- `Use $super-ai-ml-foundation to design a RAG assistant for this knowledge base.`
- `Use $super-ai-ml-foundation to choose the right model strategy for this AI feature.`
- `Use $super-ai-ml-foundation to improve this prompt and retrieval flow.`

## Evaluation Checklist

- The skill asks only for missing context.
- The output routes to the correct module.
- The answer is actionable and specific.
- Assumptions are stated clearly.
- Risks, validation steps, or next actions are included where relevant.
